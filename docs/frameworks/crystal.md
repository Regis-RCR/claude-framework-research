## The Core Idea

Crystal introduces the "Integrated Vibe Environment" (IVE) paradigm—a desktop-first approach to managing multiple AI coding sessions through visual session management and Git worktree isolation.

## Key Innovations

- **IVE Paradigm:** Desktop-first parallel development
- **Git Worktree Isolation:** Session isolation without conflicts
- **Multi-Provider Support:** Claude Code and Codex
- **Visual Session Management:** At-a-glance status monitoring

## Who Should Use This

**Perfect match:**
- Desktop-based development
- Visual workflow preference
- Multi-session management

**Probably not for you:**
- Server/headless environments
- Windows users (macOS binaries only)
- Simple single-task workflows
