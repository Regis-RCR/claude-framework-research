## The Core Idea

SystemPrompt Code Orchestrator provides a unified interface for coordinating multiple AI coding assistants. It supports both Claude Code CLI and Gemini CLI with an extensible architecture for additional agents.

## Key Innovations

- **Multi-AI Support:** Claude and Gemini (extensible)
- **MCP Integration:** systemprompt-mcp-server backend
- **Unified Interface:** Single orchestration layer across AIs
- **MIT Licensed:** Community contribution friendly

## Who Should Use This

**Perfect match:**
- Teams using multiple AI assistants
- Projects requiring AI-agnostic orchestration
- Environments exploring different AI capabilities

**Probably not for you:**
- Claude-only workflows
- Simple single-agent tasks
